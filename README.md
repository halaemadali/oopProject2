## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Navigate to src/main/java/com/hotel/Main.java
4. Right click Main.java → Run

**Requirements:**
- Java JDK 17 or higher
- IntelliJ IDEA
